I needed something to quickly stylize a simple HTML page into a readable document that looked mostly good on all devices. This is what dirtcheapcss looks like.

<style> html { color: #333; font: 300 1.2em -apple-system, sans-serif; max-width: 70ch; padding: 2ch; margin: auto; } </style>

This is clearly not meant for every solution, but in a pinch it can turn an ugly blog or documentation into a decent thing to look at.
